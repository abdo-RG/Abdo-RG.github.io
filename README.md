# Current Research

### BrainPriorNet

BrainPriorNet is my current PhD project.

The objective is to reconstruct visual stimuli directly from fMRI recordings by learning shared latent representations between brain activity and modern vision foundation models.

Main research topics include:

- Cross-subject neural decoding
- Vision-language representations
- Diffusion models
- CLIP representations
- Brain alignment
- Representation learning
