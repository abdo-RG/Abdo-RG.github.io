# Abdelilah Aitrouga — Academic Portfolio

Personal academic website, built with plain HTML/CSS/JS (no build step), deployed via GitHub Pages.

Live at: https://Abdo-RG.github.io

## Structure

```
index.html               Main page
assets/css/style.css     Styles (light/dark theme via CSS variables)
assets/js/main.js        Nav toggle, theme toggle, scroll-reveal
assets/img/profile.jpg   Headshot
assets/Abdelilah_Aitrouga_Resume.pdf   Downloadable CV
```

## Editing

Just edit `index.html` directly — each section (About, Research, Publications, Projects,
Education, Skills, Contact) is clearly marked with an HTML comment. No dependencies, no
build tools required.

## Deploying updates

Push to the `main` branch — GitHub Pages serves the repo root automatically once Pages
is enabled in the repo settings (Settings → Pages → Source: `main` branch, `/root`).
